# Preproute Admin Panel

A test management admin panel built as part of the Preproute Frontend Developer Task.

## 🔗 Live Demo
[https://preproute-admin-panel.vercel.app](https://preproute-admin-panel.vercel.app)

## Tech Stack
- React + TypeScript
- Tailwind CSS
- React Hook Form + Zod
- Axios
- Vite

## Features
- JWT-based authentication with protected routes
- Dashboard with test listing and status indicators
- Create Test with form validation
- Add Questions with rich text editor, CSV bulk import, and image URL support
- Preview and Publish flow with availability settings

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables
Create a `.env` file: VITE_API_BASE_URL=your_backend_url
